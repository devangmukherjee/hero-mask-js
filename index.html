PUT HERO MASK ON FACE written in javascript

What is so special about this?

Usually, computer vision tasks like these are performed using python and it's libraries like OpenCV. Making them harder to deploy on websites written completely in javascript (NodeJS). To bridge this prominent gap this entire program is written in javascript. It uses a library called face api-js. So that's why now programs like these are easily deployable on majority websites which are written in javascript.
